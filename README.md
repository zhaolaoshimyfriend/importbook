# 导账项目需求分析

## 项目概述
本项目用于记录导账项目的需求分析全过程，包括需求收集、分析、整理和输出。

## 文件夹结构说明

```
导账需求分析/
├── README.md                    # 项目说明文档（本文件）
├── 01-需求收集/                 # 需求收集阶段
│   ├── 会议记录/                # 会议纪要
│   ├── 访谈记录/                # 用户访谈记录
│   ├── 需求来源/                # 来自各方的需求原始材料
│   └── 问题清单/                # 待澄清的问题
├── 02-需求分析/                 # 需求分析阶段
│   ├── 业务分析/                # 业务流程分析
│   ├── 功能分析/                # 功能点分析
│   ├── 非功能需求/              # 性能、安全等非功能需求
│   └── 分析笔记/                # 分析过程中的思考记录
├── 03-需求文档/                 # 需求文档输出
│   ├── 需求规格说明书/          # 正式需求文档
│   ├── 功能清单/                # 功能列表
│   └── 需求变更记录/            # 需求变更历史
├── 04-参考资料/                 # 参考资料
│   ├── 竞品分析/                # 竞品调研
│   ├── 技术文档/                # 相关技术文档
│   ├── 业务文档/                # 业务背景资料
│   └── 其他资料/                # 其他参考资料
├── 05-沟通记录/                 # 与AI助手的沟通记录
│   └── 对话记录/                # 保存重要对话内容
└── 06-输出物/                   # 最终输出文档
    ├── 原型设计/                # 原型文件
    ├── 流程图/                  # 业务流程图
    └── 其他输出/                # 其他输出物
```

## 使用说明
1. 按照项目阶段在对应文件夹中存放相关文档
2. 重要沟通记录请保存到 `05-沟通记录/对话记录/` 中
3. 定期更新需求文档，记录变更历史
